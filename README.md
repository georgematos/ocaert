ocaert
======

Este projeto surgiu de uma ideia que tive ao começar a estudar para certificação OCAJP pelo livro da Mala Gupta, o OCA Java Programmer I Certirfication Guie Prepare. No livro há uma tabela para acompanhamento da preparação para o exame, essa tabela tem algumas particularidades. Minha ideia foi trazer essa tabela para ainternet, para que qualquer candidado ao exame possa acompanhar facilmente seu nível de preparação para o exame, de qualquer lugar em que esteja.

O projeto está disponível no meu gitHub mas ainda há muito para fazer, inclusive nesse próprio arquivo leia-me. Para e que o projeto funcione é preciso criar uma base de dados no mysql chamada oca_uer, o JPA irá gerar as tabelas necessárias a partir de então.
